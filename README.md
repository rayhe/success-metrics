# What Does Success Mean? — Cultural K-Means Clustering of Success Metrics

An interactive explorer mapping how 8 cultural clusters define success across 10 measurable dimensions, with career alignment scoring.

🔗 **[Live Site →](https://rayhe.github.io/success-metrics/)**

---

8 cultural clusters. 10 dimensions. 30 careers ranked by alignment. An interactive analysis of how different cultures define success, built on data from the World Values Survey, Hofstede, Schwartz, the OECD Better Life Index, and Deloitte's 23,000-person generational survey.

## Features

- **8 Clusters:** Asian Traditional, Western Traditional, Nordic/Scandi, Influencer/Creator, Gen Z "Gen Zen", AI Maximalist (SV), FIRE/Minimalist, Ubuntu/Communal
- **10 Dimensions:** Wealth, Education, Family/Community, Career Prestige, Health/Wellbeing, Autonomy/Freedom, Social Influence, Purpose/Impact, Security/Stability, Creative Expression
- **Interactive Career Ranker:** Select a cultural value set or build a custom one. 30 careers re-rank in real time by cosine similarity + weighted sum alignment score
- **Timeline Analysis:** 2000 → 2025 → 2030 projected value shifts across all clusters
- **Self-contained:** Single HTML file, no server needed
